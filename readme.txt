Description:
------------

NNTP-гейт для быстрого чтения форума программами просмотра ньюс-конференций.
Состоит из нескольких частей.

1. vb-nntp_core - базовый плагин для vBulletin. Отвечает за структуры данных,
   аутентификацию и администрирование.

2. vb-nntp_topics_plugin - собственно, маппинг разделов форума в NNTP-индекс
   и администрирование форумных разделов.

3. daemon - Демон NNTP-протокола. Данные о статьях забирает непосредственно их базы.
   Контроль доступа - через JSON-запросы к vb-nntp_core.


Installation:
-------------

1) vb-nntp_core
2) vb-nntp_topics_plugin

3) daemon
